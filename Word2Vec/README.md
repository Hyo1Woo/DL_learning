## Efficient Estimation of Word Representations in Vector Space (2013)
### Abstract
- 매우 큰 dataset에서 단어의 vector 표현을 위한 두 가지 모델 제안
- 단어 사이의 유사도 측정을 통해 vector 표현이 얼마나 잘 되었는지를 확인할 수 있음
- 이전 기술(NNLM)에 비해 학습시간 단축
